# ADR

This is a Python implementation of the Attribute Disclosure Risk (ADR) framework, as proposed in the paper: 'A Unified Framework for Measuring Attribute Disclosure Risks in Synthetic Data'.

## Requirements

The code is compatible with:
 - `pytorch 0.4.0`

## Run Experiments

You can find demo resources in the `demo/` directory. This folder includes:
* **Demo Data**: Sample datasets to illustrate the ADR calculation process.
* **Demo Code**: Example scripts showing how to call the ADR class and calculate scores.

Please refer to these files to understand the basic workflow of this framework.
